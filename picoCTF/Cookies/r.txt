Решение:

Задача дает нам ссылку, которая открывает веб-страницу, позволяющую нам ввести имя файла cookie. Нажав «Поиск» после ввода имени файла cookie, мы либо получим сообщение об ошибке «Похоже, это недействительный файл cookie». или мы получим сообщение об успехе, в котором говорится: «Это файл cookie! Хотя он не очень особенный...» Поскольку нам может понадобиться найдите «специальный» файл cookie, чтобы получить флаг, и мы посмотрим, что на самом деле изменяется на веб-странице при проверке наших входных данных. Подсказка из названия челленджа «Печенье». мы можем использовать расширение Google Chrome под названием editthiscookie для проверки файлов cookie на странице. Существует только один файл cookie с именем «имя», содержащий одно целое значение «0» после набрав «snickerdoodle», как подсказало нам окно поиска. Пробуя все большие значения для этого имени файла cookie, мы видим, что тип файла cookie меняется с каждым из них. Окончательно после проверки различных файлов cookie целочисленное значение «18» для файла cookie «имя» дает нам флаг.

В качестве альтернативы эту задачу можно было бы выполнить с помощью команды curl, которая передает правильное значение файла cookie, равное 18, как таковое
вводимв терминал


curl -v --cookie "name=18" http://mercury.picoctf.net:54219/check

иполучаем флаг:picoCTF{3v3ry1_l0v3s_c00k135_96cdadfd}
